# time-tracking-app

> a simple app to track time

```
$ npm run watch
```

Runs it and watches for changes, but not to index.js.

```
$ npm run build
```

Builds the app for macOS, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
